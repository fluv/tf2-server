Caretaker bot

Type !bot into chat to invoke a language model with access to the server console and the current game in context. It can see what you say in chat, but won’t process it until a message is prefixed with !bot

You can use this to modify the server configuration as needed. For example:

  • !bot add six bots
  • !bot reset the round
  • !bot send us to foundry
  • !bot turn the gravity down
  • !bot how am i playing?
  • !bot was i playing better as Scout
  • !bot adjust the team balance to make it fairer
  • !bot set host_timescale 2
  • !bot turn off random crits
  • !bot ban gordon for 15 minutes
  • !bot turn the server off
  • !bot see above
  • !bot which weapon am i using the most?
  • !bot make all the bots scout, force them to melee and constantly jump
